Color

The color palette for my design system is mainly based on an analogous color scheme and incorporates different shades of blue and purple along with a more neutral black and white. There are a total of seven colors used in my design system.

 

Blueberry

Hex Code: #40469d

Uses

The primary brand color. Is used for headings, buttons, and the logo


 

Water

Hex Code: #ededff

Uses

Main use is for the navigation bar


 

Black

Hex Code: #000000

Uses

Used for headings, text, and the Black and White versio of the logo


 

White

Hex Code: #ffffff

Uses

Used mainly as a background color. Is also present in all variations of the logo


 

Toolbox

Hex Code: #6c71b7

Uses

Used for the hover state of the primary button


 

Purple Heart

Hex Code: #5d2fa7

Uses

Used for the default state of the secondary button. Is also used for link text color


 

Lilac

Hex Code: #8c5fd6

Uses

Used for the hover state of the secondary button
