Buttons

This design system includes two button styles, a primary and secondary style.

Example

The buttons featured below are interactive. The different states may be viewed by interacting with them.

Accessibility

In order to ensure accessibility, button names should be descriptive. Buttons should not be used in place of links, and links should not be styled like buttons.


Usage

Buttons are currently used with cards which will allow the user to navigate to the different project pages.

Code Example


                  <div class="col-2"><button class="btn btn-primary">Primary</button></div>
                  <div class="col-2"><button class="btn btn-secondary">Secondary</button></div>
