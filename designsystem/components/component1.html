Navigation

The navigation bar is essential to the design system and must be included on all pages of the website. This component is comprised of a menu with links that direct the user to other pages within the site.

Example

The following navigation bar example is responsive. Adjust the browser size to view responsive behaviors.

  • Projects
  • About

Accessibility

The navigation bar uses colors from the color palette section to ensure an accessible contrast ratio. Icons should include descriptive alt attributes that inform the user where the link will direct them.

Usage

The navigation bar must be included on all pages created within this design system.

Code


                      <div class="row">
                        <div class="row">
                        <div class="col">
                          <nav class="navbar navbar-expand-lg navbar-light">
                            <div class="container-fluid">
                              <a class="navbar-brand" href="#">
                                <img src="../img/Color Logo.png" width="95" class="d-inline-block align-top" alt="Lauryn Logo"></a>
                                <button class="navbar-toggler custom-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
                                <span class="navbar-toggler-icon"></span>
                                </button>
                                <div class="collapse navbar-collapse justify-content-end nav-bg" id="navbarNav">
                                <ul class="navbar-nav">
                                <li class="nav-item">
                                <a class="nav-link active navtext" aria-current="page" href="#">Projects</a>
                                </li>
                                <li class="nav-item">
                                <a class="nav-link navtext" href="#">About</a>
                                </li>
                                </ul>

                          </div>
                          </nav>
