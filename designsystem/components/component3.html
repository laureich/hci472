Footer

The footer must be included on all pages that are created using this design system. The component contains copyright information as well as when last updates were made.

Example

The following footer is responsive. Adjust the browser size to view responsive behaviors.

© Copyright Lauryn Reich | Last updated 2022

Accessibility

The footer must use the colors from the color section to ensure a contrast ratio that meets accessibility requirements. The black text against the background is used to create high contrast for accessibility.

Usage

The footer must be included in all pages created within this design system.

Code


                            <footer class="footer-style">
                              <div class="text-center p-4">
                                <span class="lato-body">© Copyright Lauryn Reich | Last updated 2022</span>
                              </div>
                              </footer><
